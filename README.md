## 🎯 About This Repository

This repository is my **personal daily learning journal** as a Java Developer.  
Every day, one topic is added — automatically — to build consistent learning habits and maintain an active GitHub profile.

This is not just a green dot on a contribution graph.  
It's a **real, growing knowledge base** built one day at a time.

## ⚙️ How the Automation Works

```
GitHub Actions (Cron: 9:00 AM IST daily)
        │
        ▼
Calculates current day number
        │
        ▼
Appends topic content to correct .md file
        │
        ▼
Updates progress-tracker.md
        │
        ▼
Commits with message: "📚 Day N: Daily learning update - YYYY-MM-DD"
        │
        ▼
✅ GitHub contribution recorded!
```

**No server. No laptop. No manual effort.**  
Just GitHub Actions — free, always on, zero maintenance.
